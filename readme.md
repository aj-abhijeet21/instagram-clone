# Instagram clone implemented with React Native and Firebase:

Please refer to the screenshots below.

# Features

- [x] Firebase Authentication: Login, Sign Up, Sign Out
- [x] Uploading pictures via Firebase Storage
- [x] Like and commenting on Posts

# Firebase Authentication:

Users can register themselves on the platform using their email address, and then can eventually login
to the app.

# Uploading pictures via Firebase Storage:

Users can upload their pictures stored on their devices, this has been implemented using native ImagePicker; with captions as well. And then these posts can be viewed by all the users using this app.

# Like and commenting on Posts:

Users can like and comment on their posts, and then it will get updated real-time in the firebase.

# Packages used for this project:

expo<br>
firebase for upload and database functionality<br>
react-native-image-picker for using native ImagePicker<br>
formik for Form Validation<br>
